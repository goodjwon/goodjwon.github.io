# 블로그 로컬에서 띄우기
## 준비물
- Brew
- Ruby 
  - Windows 사용자 : 사이트 다운로드
  - Mac 사용자 : brew 를 통한 설치
- bundler

## 설정방법
### Mac
```
ruby -v
gem -v


brew update
rbenv install -l
rbenv install 3.1.3


gem install bundler

```

#### sh 편집
```
vi ~/.zshrc

``` sh
[[ -d ~/.rbenv  ]] && \
  export PATH=${HOME}/.rbenv/bin:${PATH} && \
  eval "$(rbenv init -)"
```

#### 정보 리로딩
``` 
source ~/.zshrc
```

#### 번들링 및 기동.(피키치 등 받기)
```
  bundle install
  bundle exec jekyll serve
```
